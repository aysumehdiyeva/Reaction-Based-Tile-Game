# 🎮 Click the Black Tile Game

A browser-based reaction game developed using **HTML, CSS, and JavaScript** for the **CTIS 255 – Frontend Web Technologies** course.

The goal of the game is to **click black tiles as quickly as possible before time runs out**. The faster the reaction time, the higher the score.

---

## 🚀 Features

- 🎯 Interactive **4×4 game board**
- ⏱ **10-second countdown timer**
- ⭐ **Dynamic scoring system based on reaction time**
- 🏆 **High score stored using LocalStorage**
- 📉 **Animated score bar**
- 🎉 **Confetti animation for new high score**
- 🎨 Smooth **CSS animations**

---

## 🕹 Gameplay

1. The game begins with a **cover screen** displaying the project members.
2. When the user clicks anywhere on the screen, a **3 → 1 countdown** begins.
3. After the countdown, the **4×4 game grid** appears.
4. **Three black tiles appear at random positions.**
5. The player must **click a black tile** to gain points.
6. The faster the player clicks, the **higher the score**.
7. The game lasts **10 seconds**.
8. When the timer reaches 0:
   - If the score beats the **high score**, a **confetti animation** appears.
   - Otherwise, a **“Time is up”** message is displayed.
